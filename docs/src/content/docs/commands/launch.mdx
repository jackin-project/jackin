## Synopsis

```bash
jackin launch [--debug]
```

<Aside type="caution">
`jackin launch` is deprecated and will be removed in a future release.
Use `jackin` (bare) or [`jackin console`](/commands/console/) instead.
The command still works — it delegates to the same operator console —
but running it prints a deprecation warning on stderr.
</Aside>

## Migration

Replace any scripted or documented invocations of `jackin launch` with
one of:

- `jackin` — opens the operator console when stdout is an interactive
  terminal, silently prints help otherwise.
- `jackin console` — always asks for the console by name; errors out if
  the terminal cannot host the TUI.

See the [operator console reference](/commands/console/) for the full
selection flow and usage details.
