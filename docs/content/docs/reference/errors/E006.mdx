**E006**: Role source not trusted

## What to try

Run `jackin config trust add <role>` to add the role to the trusted list. Only trust roles you control or have audited.

## See also

- Run `jackin doctor` for a full pre-flight health check.
- Pass `--debug` to any `jackin` command to enable additional operator output.
