**E008**: Workspace config version unsupported

## What to try

Upgrade jackin❯ (`brew upgrade jackin❯`) to a version that supports this config version.

## See also

- Run `jackin doctor` for a full pre-flight health check.
- Pass `--debug` to any `jackin` command to enable additional operator output.
