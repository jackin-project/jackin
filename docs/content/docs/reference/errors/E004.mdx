**E004**: Role manifest invalid

## What to try

Fix the syntax or schema errors shown in the error message. Run `jackin role validate <path>` to validate the manifest without launching.

## See also

- Run `jackin doctor` for a full pre-flight health check.
- Pass `--debug` to any `jackin` command to enable additional operator output.
