**E002**: Docker version too old

## What to try

Upgrade Docker to the latest stable release from [docker.com](https://www.docker.com/). Run `docker version` to see the current server version.

## See also

- Run `jackin doctor` for a full pre-flight health check.
- Pass `--debug` to any `jackin` command to enable additional operator output.
