**E011**: Port conflict on DinD TLS port

## What to try

Stop the process using the conflicting port, or restart the DinD network stack. Run `jackin prune` if the port is held by a stale instance.

## See also

- Run `jackin doctor` for a full pre-flight health check.
- Pass `--debug` to any `jackin` command to enable additional operator output.
