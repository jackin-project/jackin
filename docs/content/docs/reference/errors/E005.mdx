**E005**: Role manifest version unsupported

## What to try

Upgrade jackin❯ (`brew upgrade jackin❯`) to support the manifest version, or pin the role to a manifest version this binary supports.

## See also

- Run `jackin doctor` for a full pre-flight health check.
- Pass `--debug` to any `jackin` command to enable additional operator output.
