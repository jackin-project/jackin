**E010**: DinD sidecar failed health check

## What to try

Run with `--debug` for additional operator output. If OTLP export is configured, share the invocation ID when reporting the failure. Try `jackin purge` to clean up the stale instance and re-run `jackin load`.

## See also

- Run `jackin doctor` for a full pre-flight health check.
- Pass `--debug` to any `jackin` command to enable additional operator output.
