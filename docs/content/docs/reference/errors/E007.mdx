**E007**: Workspace not found

## What to try

Run `jackin workspace list` to see saved workspaces. Use `jackin workspace create` to add a new one.

## See also

- Run `jackin doctor` for a full pre-flight health check.
- Pass `--debug` to any `jackin` command to enable additional operator output.
