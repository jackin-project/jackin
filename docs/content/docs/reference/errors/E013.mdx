**E013**: 1Password CLI not signed in

## What to try

Run `op signin` to sign in to 1Password CLI, or remove `op://` references from your workspace env vars if you are not using 1Password.

## See also

- Run `jackin doctor` for a full pre-flight health check.
- Pass `--debug` to any `jackin` command to enable additional operator output.
