title: "CI Matrix Split"
---

**Status**: Open — design proposal

## Problem

<RepoFile path=".github/workflows/ci.yml">.github/workflows/ci.yml</RepoFile> has a single `check` job that runs fmt + clippy + `cargo nextest run --all-features` + the real-Docker E2E (`dind_e2e`). As the codebase grows, this becomes a multi-minute job where one failing E2E masks unit-test signal and vice versa. The construct-image rebuild path adds 5–10 minutes on arm64 when the construct changes.

Evidence: <RepoFile path="crates/jackin/tests/dind_e2e.rs">crates/jackin/tests/dind_e2e.rs</RepoFile> is 1094 LOC and exercises real `docker run`. The construct-image rebuild path adds significant time on arm64 when Dockerfiles change.

## Proposal

Split into a matrix of jobs:

| Job | What it runs |
|---|---|
| `lint` | fmt + clippy + `cargo check --all-targets` |
| `unit` | `cargo nextest run --lib --bins` |
| `integration` | all tests except E2E |
| `e2e-docker` | real Docker E2E — only when `docker/**` or `tests/dind_e2e.rs` changes |
| `capsule` | builds + tests `crates/jackin-capsule/` in isolation |
| `msrv` | as today |
| `build-validator` | as today |
| `ci-required` | aggregator |

Use `cargo nextest run --partition` for parallelism within `unit`/`integration` if needed.

## Non-goals

- Do not add a bench job yet.
- Do not change the sccache / mold / cargo-registry-cache stanzas beyond what the split requires.
- Do not restructure the construct-image build trigger logic — keep the existing path-filter pattern.

## Implementation Phases

### Phase 1 — Split check into lint + unit + integration + e2e-docker

Direct refactor of `ci.yml`. The existing `ci-required` aggregator pattern composes cleanly. Preserve sccache / mold / cargo-registry-cache stanzas.

### Phase 2 — Add capsule job

Build and test `crates/jackin-capsule/` in isolation. Only runs when `crates/jackin-capsule/**` changes.

### Phase 3 — Path-aware triggers

Wire path-filter logic so `e2e-docker` only runs when `docker/**` or `tests/dind_e2e.rs` changes. Capsule job only when its crate changes.
