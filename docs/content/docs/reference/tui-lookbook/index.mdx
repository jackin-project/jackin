title: TUI Lookbook
description: Rendered previews for shared jackin-tui components
---

The TUI lookbook is the contributor-facing catalog for shared `jackin-tui`
components. Each preview is generated by
<RepoFile path="crates/jackin-tui/src/bin/tui-lookbook.rs" /> from the real
component stories in <RepoFile path="crates/jackin-tui/src/lookbook.rs" /> using
Ratatui's `TestBackend`, then committed under `docs/public/tui-lookbook/`.

Regenerate the previews after changing a shared component:

```bash
cargo run -p jackin-tui --bin tui-lookbook -- docs/public/tui-lookbook
```

Verify the committed previews without rewriting them:

```bash
cargo run -p jackin-tui --bin tui-lookbook -- --check docs/public/tui-lookbook
```
